## 🚀 프로젝트 개요

작업자가 기본 정보만 입력하면 **AI가 안전조치·위험요인·작업절차**를 자동 제안하는 **안전작업허가서 생성 서비스**입니다.
과거 사고사례 **RAG**, 실시간 날씨(**OpenWeather API**), **n8n 워크플로우**를 결합하여 작성 시간을 단축하고 현장 안전성을 높입니다.

## 🧩 주요 기능

* 입력: 작업유형/장소/설비/기간/작업내용 등
* AI 자동 생성:

  * **안전조치사항**
  * **위험요인 & 감소대책**
  * **작업절차(체크리스트)**
  * **기상정보(기온·습도·풍속·상태) 및 작업자 건강상태 자동 반영**
* 결과 **검토/승인** 절차 포함

## 🏗 기술 구성

| 구분         | 기술                              |
| ---------- | ------------------------------- |
| 프론트        | HTML / JS / CSS (Replit Web UI) |
| 백엔드        | Node.js (Express, Replit)       |
| AI 오케스트레이션 | n8n 워크플로우                       |
| 지식검색       | RAG (사고사례 엑셀 → 임베딩·벡터검색)        |
| 외부 데이터     | OpenWeatherMap (실시간 날씨)         |
